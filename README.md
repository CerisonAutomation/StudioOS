# StudioOS

**Professional interior design project management platform** built for studio teams managing client projects, scheduling, and deliverables.

---

## What it does

- Project and client management for interior design studios
- Task tracking, scheduling, and milestone management
- File and asset management for design deliverables
- Team collaboration and role-based access
- Clean, modern UI optimised for creative professionals

---

## Stack

```
Frontend    TypeScript · React · shadcn/ui · Tailwind CSS
Backend     Supabase · PostgreSQL · REST APIs
Auth        Supabase auth with RLS policies
```
